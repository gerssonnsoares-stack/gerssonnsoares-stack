# Meu Portfólio Profissional 🚀

Este é o meu primeiro portfólio profissional desenvolvido como parte das atividades do curso de **Sistemas para Internet** da UAP, no município de Brasileira - PI.

O projeto apresenta minha biografia, objetivos profissionais e a renderização dinâmica de projetos utilizando JavaScript.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica do site.
* **CSS3:** Estilização moderna com uso de Flexbox e CSS Grid para responsividade.
* **JavaScript (ES6):** Manipulação do DOM para renderização dinâmica de dados a partir de um array de objetos.
* **GitHub Pages:** Hospedagem e publicação do projeto.

## 👤 Sobre Mim

Sou **Gerson do Nascimento Soares**, natural de Piripiri/PI. Atualmente curso Sistemas para Internet e busco me qualificar para o mercado de trabalho, com foco em desenvolvimento web e crescimento profissional.
